A Repository containing the solution to Syngenta's selective process
===

To install the required modules:

```
$ pip install -r requirements.txt
```
or
```
$ pip3 install -r requirements.txt
```


To run the test:

```
$ py.test
```

To run the main file go to src/:

```
$ cd src/
```

Then run:

```
$ python3 __init__.py
```

or

```
$ python __init__.py
```
